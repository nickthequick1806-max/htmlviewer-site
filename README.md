# HTML Viewer

A modern, browser-based HTML editor built for creating, testing, validating, and previewing websites in real time.

🌐 **Live Website:** [htmlviewer.site](https://htmlviewer.site)

## Overview

HTML Viewer provides a complete web development workspace directly in your browser. Write and preview HTML code, detect validation issues, use AI-assisted editing tools, manage reusable presets, and save your work locally without installing additional software.

The editor is designed to make HTML development faster, easier, and safer—whether you are testing a small code snippet or building a complete webpage.

## Features

### AI-Powered HTML Editor

Use the built-in AI assistant to generate, edit, improve, explain, and troubleshoot HTML, CSS, and JavaScript code.

### Real-Time Live Preview

See changes instantly while editing your code. Preview layouts across desktop, tablet, and mobile screen sizes.

### Built-In HTML Validation

Automatically detect invalid markup, unexpected elements, missing tags, and other common HTML problems.

Validation errors are displayed alongside the affected lines to help you quickly identify and fix issues.

### Local Code Saving

Your projects and saved versions are stored locally in your browser, helping protect your work from accidental page refreshes, browser closures, or lost sessions.

No account is required to save code locally.

> Clearing your browser storage or using a different browser or device may remove locally saved data.

### Presets

Save reusable code presets and quickly load them back into the editor whenever needed.

Community presets can also be used to discover layouts, components, and starting points created for the HTML Viewer platform.

### Code Formatting

Beautify messy code for improved readability or minify it for a smaller and more compact output.

### Search and Replace

Quickly locate and replace code using options such as case-sensitive matching and whole-word searching.

### Project Exporting

Export your work as an HTML file and download it directly to your device.

### Responsive Interface

HTML Viewer is designed to work across desktop and mobile devices with a clean, modern, onyx-black interface.

## Getting Started

1. Visit [htmlviewer.site](https://htmlviewer.site).
2. Enter or paste your HTML code into the editor.
3. View the rendered result in the live preview.
4. Use the validation tools to detect code issues.
5. Save your code locally or create reusable presets.
6. Use the AI editor when you need help generating, updating, or debugging code.

## Use Cases

HTML Viewer is useful for:

* Testing HTML, CSS, and JavaScript snippets
* Building and previewing webpages
* Debugging invalid HTML
* Learning web development
* Creating reusable website layouts
* Editing code with AI assistance
* Previewing responsive designs
* Saving multiple versions of a project

## Privacy

HTML Viewer is designed with privacy in mind. Locally saved projects remain inside your browser unless you choose to export, share, or submit them through another feature.

Avoid placing passwords, private API keys, access tokens, or other sensitive credentials inside client-side code.
